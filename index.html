Jak funguje tato statická webová stránka?

Tato stránka je vytvořena pouze pomocí HTML, CSS a JavaScriptu. Nevyužívá žádný server – veškerý obsah je přímo ve zdrojovém kódu.

  • <header> – záhlaví stránky
  • <section> – obsahové bloky jako výběr rtěnek, produkty, košík
  • JavaScript – pro filtrování produktů a přidávání do košíku

Žáci si mohou stránku upravit, změnit produkty nebo barvy a hned vidět výsledek.
